# 將棋遊戲網站工作計畫

更新時間：2026-03-16

## 目標
建立一個日式風格的將棋網站，支援完整將棋規則、單人對戰（自我對弈與對電腦）、以及網際網路線上雙人對戰，並提供清楚的勝負回饋。

## 工作SOP（勾選清單）

- [x] 釐清需求與功能範圍
- [x] 盤面與棋子資料結構設計（含成、持駒、打入）
- [x] 規則引擎（合法走子、王手/將死、千日手、持將棋）
- [x] 單機模式：自我對弈（同機兩人）
- [x] 單機模式：對電腦（基礎AI/搜尋）
- [x] 介面設計：日式視覺風格（木紋、和風色系、字體）
- [x] 介面互動：拖放、可走提示、非法走子提示
- [x] 勝負回饋：將死/投了/千日手/時間到（視規則配置）
- [x] 線上對戰：配對/房間/斷線重連
- [x] 線上同步：回合狀態、走子驗證、延遲處理
- [ ] 測試：規則單元測試、整合測試
- [ ] 部署：可公開訪問的線上環境

## 交付物
- 可在瀏覽器執行的將棋網站
- 完整規則與勝負判定
- 單機對戰（自我對弈與AI）
- 線上雙人對戰
- 日式風格視覺
